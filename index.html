Calculadora De Contas

Em resumo, este projeto foi atualizado para permitir que o usuário adicione informações sobre seu salário. A página apresenta um formulário separado para adicionar as informações, e JavaScript é usado para calcular o valor total das contas e subtrair esse valor do salário mensal do usuário. O resultado é exibido em um elemento de saída, indicando se o usuário tem dinheiro suficiente para pagar todas as contas ou quanto falta para pagar todas as contas.
